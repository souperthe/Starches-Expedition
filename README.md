# STARCHES EXPEDITION

a very simple Rayman inspired 2d platformer I had created in [Godot 4.6](https://godotengine.org/releases/4.6/) as a Personal Interest project for school

## FEATURES

- keyboard and controller support
- scalable local co-op, requires extra bindings though
- scalable state machine system for the player
- music and score global managers
- project files! this contains most if not all the flas and flps i've used for this project

## PLUGINS USED

- console : https://github.com/jitspoe/godot-console
